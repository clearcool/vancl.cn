三、本商城是携带有商家入驻功能，经营商品分为自营商品和商家商品。

四、本网站下载源码后，只需修改 .env 文件内的数据库配置即可连接到
    本地数据库，即可使用。
	
    本项目数据库、数据表关系E-R图、思维导图均在	introduction 文
    件夹内。	
   
    注：在数据库迁移过程中 user_shop 表中的商家 us_id 可能自动
    发生改变，结果会导致商家商品遍历不出，本项目默认自营的店铺
    us_id 为0，如有更改修改数据库即可，sname为Vancl的字段为
    自营字段。（其他店铺也可能更改，删掉重新建立新店铺即可）
